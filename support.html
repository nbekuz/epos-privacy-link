Support URL (App Store)

Рекомендуемые значения для поля Support URL в App Store Connect. Указывайте только ссылки, которые открываются в браузере по протоколу https://.

В приложении на экране «Контакты» есть действия, которые открывают не веб-страницу (tel:, max://, tg:// и т.п.) — такие схемы нельзя вставлять в поля URL в App Store. Телефон, MAX, прямые ссылки в мессенджеры остаются внутри приложения; в Store указываются только https:// адреса.

Основной вариант

Официальный сайт — тот же ресурс, что открывается по кнопке «эпос.com» в разделе «Контакты» (внутри приложения — во встроенном браузере). Для Support URL логично указать именно его:

https://xn--n1abh9b.com

Это punycode-представление домена эпос.com; по смыслу это один и тот же сайт.

Альтернатива

Если сайт пока минимальный, а основная оперативная поддержка идёт через открытый Telegram-канал, допустимо указать его как Support URL (или продублировать логику в примечаниях к приложению):

https://t.me/epos63

Кратко

  • Рекомендуется: https://xn--n1abh9b.com
  • При необходимости: https://t.me/epos63
